# Bhejo

**Bhejo** (from the Punjabi word *bhejo*, meaning "to send") is a lightweight social media cross-posting tool.

## Goal
To provide a simple interface for quickly posting content to multiple social media platforms simultaneously.

## Philosophy
- **Not a Social Network**: This is a utility, not a destination.
- **Simplicity**: Input content once, distribute everywhere.
- **Speed**: Designed for quick updates.

## Planned Features
- [ ] Multi-platform support (Twitter/X, LinkedIn, Mastodon, Bluesky, etc.)
- [ ] Unified composer interface
